# AniShelf

A RESTful Anime & Manga Library API that allows users to manage personal collections,
track reading progress, and search titles using external APIs.

## Tech Stack
- Node.js, Express
- SQLite
- JWT Authentication
- AniList GraphQL API
- Jikan REST API

## Features
- User authentication and authorization (JWT)
- CRUD operations for anime/manga collections
- Pagination and input validation
- External API integration for metadata
- Swagger API documentation

## Why I Built This
I wanted to practice building a real-world backend service with authentication,
third-party APIs, and clean REST design.
